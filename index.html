UI/UX Design

UI (user interface) design is all about what users actually see on the application’s screen – text, colors, backgrounds, icons, and moving elements like animations.
UX (user experience) design focuses on studying how users move through all the different elements of the UI. For UX designers, it’s all about the flow of the user experience and eliminating any sources of friction during these interactions.
